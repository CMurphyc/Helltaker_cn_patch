# Helltaker_cn_patch

### 简介
地狱把妹王汉化补丁（个人汉化）

### 如何使用
将本repo clone到本地后把local_cn_patch文件夹的名字修改为local, 找到helltaker本地目录, 把local文件夹直接扔进去覆盖掉就好

ps.建议覆盖前备份游戏本地的local文件夹
